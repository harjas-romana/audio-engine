## what is this thing??

ok so basically i was tired of youtube audio sounding mid on my earbuds. so i made this extension that lets u mess with the audio in realtime. like u can add surround sound, boost the bass, make vocals clearer, all that good stuff

its got like a bunch of modes n settings n u can save ur own presets too which is pretty cool ngl

---

## features n stuff

- **16D surround sound** - makes it sound like the music is all around u (not the rotating kind, just fixed surround)
- **360 degree audio** - full sphere immersion thing, sounds crazy with good earbuds
- **studio mode** - for when u want that clean flat sound, no coloring
- **concert hall** - makes it feel like ur in a big room lol
- **vocal clarity** - pushes the vocals forward if u wanna hear lyrics better

theres also:
- 10 band equalizer (the sliders thing)
- bass n treble buttons for quick adjustments
- reverb controls
- stereo width (make it wider or mono)
- compressor n bunch of other pro stuff
- save ur own presets!!
- factory presets if ur lazy like me lmao

## how to install

1. download all the files n put em in a folder called `audio-engine` or watever u want
2. make sure theres a `icons` folder inside with the icon images
3. open ur browser (chrome, dia, edge, brave, any chromium one works)
4. go to `chrome://extensions` in the url bar
5. turn ON "developer mode" - theres a toggle on the top right
6. click "load unpacked" 
7. select the folder u made
8. done!! u should see the icon in ur toolbar now

---

## how to use

1. go to youtube.com
2. play any video/music
3. click the extension icon
4. turn the master switch ON
5. **important**: click anywhere on the youtube page once (this activates the audio stuff, browser security thing idk)
6. now just mess with the settings n it changes in realtime, no need to refresh or anything

## troubleshooting

**audio not changing?**
- make sure master toggle is ON
- click somewhere on the youtube page
- try refreshing the page n turning it on again

**no sound at all??**
- turn it off n on again lol
- check if the video is actually playing
- check master volume slider isnt at 0

**extension not showing?**
- make sure u loaded it in developer mode
- check if its enabled in extensions page
